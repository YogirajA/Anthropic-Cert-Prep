Section two, quick review.

Model lineup (Jun 2026). claude-fable-5 / claude-mythos-5 (most capable; Mythos = Project Glasswing) then claude-opus-4-8 (default) then claude-opus-4-7, claude-opus-4-6, claude-sonnet-4-6, claude-haiku-4-5. Mythos Preview retires Jun 30 2026. Exam does NOT test model versions — context only.

Fable 5 / Mythos 5. Thinking always on (omit param). New tokenizer about 30% more tokens (started Opus 4.7). stop_reason: "refusal" (HTTP 200 — check before reading content). Refusal fallbacks param retries on another model. No ZDR (30-day retention).

Task Budgets (beta). task-budgets-2026-03-13. output_config.task_budget:{type:"tokens",total:N} (min 20k) — model sees a running countdown for the whole loop. Distinct from max_tokens (per-response cap, invisible to model). Fable 5 / Opus 4.7 / 4.8.

Mid-conversation system. Append {role:"system"} inside messages[] to update operator instructions mid-task without breaking the cached prefix. Opus 4.8; no beta header per current docs.

Output token caps. Opus 4.8/4.7/4.6 and Fable 5 / Mythos 5 = 128k. Sonnet 4.6 / Haiku 4.5 = 64k. Batch with output-300k-2026-03-24 beta = 300k.

SDK + tool renames. "Claude Code SDK" then "Claude Agent SDK". Task tool then Agent tool (v2.1.63). Both names coexist.

Skills replace commands. Custom slash commands merged into Skills. .claude/skills/<name>/SKILL.md canonical. Old .claude/commands/ still works.

Plugins. New packaging unit. Bundle skills, agents, hooks, MCP, output styles. Distribute via marketplace, git URL, zip.

Adaptive thinking. thinking: {type: 'adaptive'} + output_config.effort (low/medium/high/xhigh/max). Replaces budget_tokens on 4.6/4.7.

xhigh effort. New top tier on Opus 4.7/4.8 and Fable 5 / Mythos 5. Recommended default for coding/agentic.

Prefill deprecated. 4.6 / 4.7 / 4.8 and Fable 5 / Mythos 5 no longer support last-assistant-turn prefill. Use Structured Outputs / strict tools / XML.

Structured Outputs GA. output_config.format. No beta header. SDK helpers in Python (Pydantic), TS (Zod), Java. Incompatible with Citations.

Memory tool GA. memory_20250818, no beta header. Client-side, /memories backend, 6 commands.

Compaction beta. compact-2026-01-12. Server summarizes; Opus 4.8/4.7/4.6, Sonnet 4.6, Fable 5 / Mythos 5.

Auto prompt caching. Top-level cache_control: {type:'ephemeral'} auto-moves breakpoint to last cacheable block.

1M context standard. Opus 4.8/4.7/4.6, Sonnet 4.6, Fable 5 / Mythos 5: no beta, no premium pricing. Haiku 4.5 = 200k.

--bare CI mode. Skips local config (hooks/skills/plugins/MCP/auto-memory/CLAUDE.md). Recommended CI; future -p default.

dontAsk permission. Strictest mode: denies anything not in permissions.allow. Use for CI hardening.
